---

## 📋 О проекте

Проект реализует два мощных эвристических алгоритма для решения классической **задачи коммивояжера (TSP)**:

- 🧬 **Генетический алгоритм** — эволюционный подход с популяцией решений
- 🔥 **Имитация отжига** — вероятностный метод, вдохновлённый физикой

Приложение позволяет не только наблюдать за процессом оптимизации в реальном времени, но и проводить сравнительное тестирование алгоритмов на разных размерностях задачи.

## ⚙ Параметры алгоритмов

### Генетический алгоритм

| Параметр | Значение | Описание |
|----------|:--------:|----------|
| Размер популяции | 100 | Количество маршрутов в одном поколении |
| Количество поколений | 500 | Максимальное число итераций эволюции |
| Вероятность мутации | 2% | Шанс случайного обмена двух городов в маршруте |
| Элитизм | 10% | Лучшие маршруты гарантированно переходят в следующее поколение |
| Размер турнира | 5 | Количество кандидатов при отборе родителей |

**Дополнительные параметры:**
- Тип кроссинговера: частично-отображенный (PMX)
- Тип селекции: турнирная
- Функция приспособленности: fitness = 1 / расстояние

### Имитация отжига

| Параметр | Значение | Описание |
|----------|:--------:|----------|
| Начальная температура | 1000 | Определяет вероятность принятия ухудшающих решений |
| Коэффициент охлаждения | 0.995 | Скорость уменьшения температуры (T = α × T) |
| Максимальное число итераций | 50000 | Длительность процесса поиска |

**Закон охлаждения:**
Tₖ₊₁ = α × Tₖ, где α = 0.995

**Вероятность принятия решения:**
P = exp(-ΔE / T), где ΔE = L(P*) - L(P)

## 🖥 Интерфейс программы

### Главное окно

Приложение состоит из двух основных вкладок: **«Визуализация»** и **«Сравнение результатов»**.

[Схема расположения элементов]

Верхняя часть окна — переключатель вкладок.
Центральная часть — основное рабочее поле.
Нижняя часть — панель управления с кнопками.

---

### Вкладка «Визуализация»

#### Основное поле (карта маршрута)

| Элемент | Описание |
|---------|----------|
| 🔵 Синие круги с номерами | Города, которые нужно посетить |
| 🌈 Цветные линии | Текущий маршрут коммивояжёра (цвет меняется от красного к зелёному по мере улучшения) |
| ➡️ Красные стрелки | Направление движения по маршруту |
| 🔘 Серая сетка | Вспомогательная координатная сетка (шаг 50 пикселей) |

#### Информационная панель (левый верхний угол)

[Прямоугольная полупрозрачная панель]

Содержит:
- Название текущего алгоритма
- Номер текущего шага / общее количество шагов
- Текущее расстояние маршрута
- Количество городов
- Индикатор прогресса (заполняющаяся полоска)
- Количество кадров анимации

#### Панель управления (внизу)

| Кнопка | Действие |
|--------|----------|
| 🧬 Генетический алгоритм | Запускает ГА с текущими параметрами |
| 🔥 Имитация отжига | Запускает SA с текущими параметрами |
| 🔄 Сброс (новый граф) | Генерирует новый случайный набор городов |
| ▶ Воспроизвести анимацию | Проигрывает сохранённые кадры оптимизации |
| ⏹ Остановить | Останавливает анимацию или алгоритм |
| 💾 Экспорт маршрута | Сохраняет лучший маршрут в текстовый файл |
| ⚡ Скорость | Регулирует скорость анимации (5 уровней: очень быстрая, быстрая, средняя, медленная, очень медленная) |

#### График сходимости (справа)

- Синяя линия — улучшение маршрута генетическим алгоритмом
- Красная линия — улучшение маршрута имитацией отжига
- Зелёная метка — лучшее достигнутое расстояние
- Красная точка — текущее положение на графике

---

### Вкладка «Сравнение результатов»

#### Панель управления

| Элемент | Описание |
|---------|----------|
| 🔽 Количество городов | Выбор размерности задачи (20, 30 или 50) |
| 🗺 Просмотреть полный граф | Открывает отдельное окно со всеми возможными связями |
| ▶ Запустить тесты | Выполняет 5 запусков каждого алгоритма и собирает статистику |
| 📄 Экспорт результатов | Сохраняет таблицу и график в HTML-файл |
| 🗑 Очистить | Удаляет текущие результаты из таблицы и графика |

#### Таблица результатов

Таблица содержит следующие столбцы:
- Городов (20, 30, 50)
- Алгоритм (Генетический / Имитация отжига)
- Лучшее расстояние (минимальное за 5 запусков)
- Среднее расстояние (среднее арифметическое)
- Среднее время (мс)
- Стабильность (в процентах, показывает разброс результатов)

#### График сходимости (вкладка сравнения)

- Отображает усреднённые графики сходимости для выбранного количества городов
- Позволяет визуально сравнить скорость и качество оптимизации
- Синяя линия — Генетический алгоритм
- Красная линия — Имитация отжига

#### Лог выполнения

Текстовая область, где отображается процесс тестирования:
- Заголовок с количеством городов
- Информация о каждом запуске (расстояние, время)
- Итоговая статистика по каждому алгоритму

---

### Окно «Полный граф»

Открывается при нажатии кнопки **«🗺 Просмотреть полный граф»**.

В этом окне:
- Серыми линиями показаны все возможные пути между городами
- Синими кругами обозначены города с их номерами
- Заголовок окна содержит количество городов
